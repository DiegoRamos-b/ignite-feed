## 🚀 Instalando Ignite Feed

Para instalar o Ignite Feed, siga estas etapas:

</br>

**Linux, macOS e Windows :**

**Clone o repositório**

```
  git clone https://github.com/rms-diego/ignite-feed.git
```

Depois execute os seguintes comando

**Npm**

```
  npm i && npm run dev
```
